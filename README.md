    <h1>🌦️ KnowYourWeather</h1>
    <p>
      <b>KnowYourWeather</b> is a simple yet powerful weather forecasting web application. 
      Just enter the <b>name of the city</b> and instantly get real-time weather updates such as 
      temperature, humidity, and conditions — all displayed in a clean and interactive interface.
    </p>
    
    <h2>✨ Features</h2>
    <ul>
      <li>🌍 Enter any city name to get instant weather updates</li>
      <li>🌡️ Real-time temperature, humidity, and condition display</li>
      <li>⚡ Fast performance powered by React + Vite</li>
      <li>🎨 Modern and responsive UI for all devices</li>
    </ul>
    
    <h2>🚀 Future Developments</h2>
    <ul>
      <li>📅 5-day and 7-day forecast support</li>
      <li>📍 Auto-detect location using GPS</li>
      <li>🌙 Dark/Light mode support</li>
      <li>🗺️ Interactive weather maps</li>
      <li>📊 Historical weather data visualization</li>
    </ul>
